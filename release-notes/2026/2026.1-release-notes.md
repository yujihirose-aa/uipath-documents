# UiPath 2026.1 リリースノート

> 出典: https://docs.uipath.com/release-notes/docs/january-2026
> 取得日: 2026年4月17日

## 🤖 Agents & Maestro

### Maestro（エージェントオーケストレーション）
- **Goal Based Agents** の強化：複数エージェントの協調動作が改善
- エージェント間のコンテキスト共有機能の追加
- Maestro ダッシュボードでのリアルタイム監視機能

### UiPath Agents
- LLM連携エージェントのパフォーマンス向上
- エージェントのデバッグ・テスト機能の強化
- カスタムエージェントテンプレートの追加

## 🏭 Orchestrator

- キュー管理UIの刷新
- SLA監視・アラート機能の強化
- マルチテナント環境でのパフォーマンス改善
- Automation Hub との統合強化

## 📄 Document Understanding

- 新しい事前学習済みモデルの追加（請求書・契約書・医療文書）
- OCR精度の向上（日本語・アジア言語対応強化）
- Generative AI を活用したデータ抽出精度の向上
- Action Center との連携改善

## 🧪 Test Suite

- テスト自動化のAI支援機能（テストケース自動生成）
- CI/CDパイプライン統合の強化
- モバイルテスト機能の拡張

## ☁️ Automation Cloud

- クラウドインフラのスケーラビリティ向上
- セキュリティ強化（ゼロトラストアーキテクチャ対応）
- 新リージョン追加（アジア太平洋）

## 🔧 Studio / StudioX

- 新しいアクティビティパッケージの追加
- Generative Recorder の精度向上
- デバッグ体験の改善
- .NET 8 対応強化
